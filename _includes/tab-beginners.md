## CYBEROWL

Це офіційний застосунок від Кібер Корпусу України для операційних систем з графічним інтерфейсом, який містить основні інструменти для DDOS-атак на російські ресурси.

Цей застосунок розрахований на різний рівень знань в ІТ та підходить як для звичайних користувачів, так і більш досвідчених айтішників, має можливість працювати у фоні, згортатися в трей при закритті, має автозавантаження та автооновлення.

Налаштування можна залишити за замовчуванням або керувати модулями DDOS-інструментів, які автоматично завантажують цілі ворожих ресурсів та використовують методи атак Кібер Корпусу України.

## 1. Процес встановлення {#процес-встановлення}

Завантажити застосунок для своєї платформи можливо за посиланням [github.com/corpus-dev/cyberowl/releases](https://github.com/corpus-dev/cyberowl/releases)

1. Перший етап встановлення – Вибір мови

   ![Вибір мови]({{ site.baseurl }}/assets/img/sova/image1.png)

   Обираємо якою мовою нам приємніше бачити інтерфейс і їдемо далі

2. Обираємо куди будуть зберігатись встановленні модулі для нашої атаки, щоб не плутатись залишаємо за замовчуванням

   ![Вибір директорії модулів]({{ site.baseurl }}/assets/img/sova/image2.png)

3. Вводимо свій CyberCORPS ID який можна отримати по інструкції за посиланням [@corps_statistics_bot](https://t.me/corps_statistics_bot)

   ![Введення CyberCORPS ID]({{ site.baseurl }}/assets/img/sova/image3.png)

4. Останній за списком, але не за значенням: Вибір стартових налаштувань (пресету)

   Розберемо їх:

   ![Вибір пресету]({{ site.baseurl }}/assets/img/sova/image4.png)

- Пресети які автоматично завантажують та запускають один з модулів атаки:
  - Державна установа / старий ПК – встановлює мінімальні параметри використання модуля для запуску їх на слабких машинах та/або використання слабкої/завантаженої мережі. Бажано використовувати на роботі, щоб не заважати іншим працівникам
  - Ноутбук – розрахований на використання мобільного заліза який має доступ до інтернету за допомогою WI-Fi
  - Комфортний – обирає невисокі налаштування щоб не заважати комфортній роботі з комп'ютером
  - Нормальний – встановлює стандартні значення атаки
  - Максимальний – автоматично підлаштовує модулі під максимально можливі параметри атаки з вашого комп'ютера, може призвести до суттєвого зменшення продуктивності інших програм
  - Експерт – Встановлює виключно графічну частину програми дозволяючи самостійно обрати бажаний модуль та налаштувати конфігурацію атаки (рекомендовано виключно для досвідчених користувачів)

Вітаю! Ви успішно встановили на свій комп'ютер офіційний програмний комплекс Cyber Owl та вступили до спільноти Кіберкорпусу.

## 2. Інтерфейс програми {#інтерфейс}

### 2.1 Головне вікно {#головне-вікно}

![Головне вікно програми]({{ site.baseurl }}/assets/img/sova/image5.png)

Умовно можна розділити на 4 блоки, такі як:

- Меню навігації
- Загальна інформація
- Графічне відображення роботи модуля
- Журнал виконання

Розглянемо їх більш детально:

- Меню навігації – надає доступ до інших сторінок програми та базові системні налаштування
- Загальна інформація – містить в собі базову інформацію по роботі програми таку як: назва та статус модуля який використовується на даний момент, статистика використаного трафіку мережевого інтерфейсу, ваш Corpus ID у скороченому вигляді та поточну кількість підключень
- Графічне відображення роботи модуля – графік використання трафіку за вказаний проміжок часу
- Журнал виконання – базовий вивід роботи інструмента

### Модулі {#модулі}

![Сторінка модулів]({{ site.baseurl }}/assets/img/sova/image6.png)

На цій сторінці ми можемо отримати доступ до налаштувань модулів атак

- Активний модуль – дає можливість обрати модуль для атаки та ввімкнути/вимкнути її за потреби
- Доступні модулі – перелік модулів які можна задіяти та доступ до їх тонких налаштувань.

**Не рекомендовано змінювати налаштування вручну якщо ви не експерт!**

**MHDDOS PROXY**

- Не потребує VPN (умовно) - автоматично завантажує і підбирає робочі проксі для заданих цілей, періодично їх оновлюючи
- Атака декількох цілей одночасно з автоматичним балансуванням навантаження
- Використовує різні методи для атаки та змінює їх в процесі роботи

![MHDDOS PROXY]({{ site.baseurl }}/assets/img/sova/image7.png)

**DISTRESS**

- Не потребує VPN (умовно) - автоматично завантажує і підбирає робочі проксі для заданих цілей, періодично їх оновлюючи
- Атака декількох цілей одночасно з автоматичним балансуванням навантаження
- Використовує різні методи для атаки та змінює їх в процесі роботи

### Налаштування {#налаштування}

Налаштування безпосередньо програмного комплексу СyberOwl, такі як:

- Автоматичне оновлення комплексу до останньої версії
- Автоматичний запуск програми при запуску комп'ютера
- Прибирати програмну в трей при запуску комп'ютера та натискані на кнопку вихід у вікні програми
- Зміна мови
- Зміна вашого Corpus ID
- Зміна вашого Corpus API Key
- Вибір теми (світла/темна)
- Отримання доступу та зміна розташування безпосередньо теки з встановленими модулями
- Видалення даних модулів або програмного комплексу в цілому

![Налаштування]({{ site.baseurl }}/assets/img/sova/image9.png)
